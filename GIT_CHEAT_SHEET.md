# Git Cheat Sheet forletaheader

## 1. Check Status (The "Where am I?" command)
Run this to see what files have changed.
```bash
git status
```

## 2. Save Changes (The "Save Point" combo)
Run these two commands together to save your work to history.
```bash
git add .
git commit -m "Description of what I did"
```

## 3. Upload to Cloud
Run this to send your saves to GitHub.
```bash
git push
```
